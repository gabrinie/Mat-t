A FUNÇÃO QUADRÁTICA

A Função Quadrática é um cálculo matemático que busca encontrar uma incógnita dentro de uma expressão em forma de um polinômio de grau dois, o que significa que a incógnita está distribuída de forma que aparece elevada ao quadrado. Sua estrutura é determinada pela seguinte expressão: f(x) = ax² + bx + c, onde os termos a, b, e c são os coeficientes do cálculo da função, e por regra o coeficiente a não deve possuir valor igual a zero. No estudo da álgebra entendemos que o grau de uma função é determinado pelo número de valores (raízes) que essa função pode apresentar, dessa forma a partir dos cálculos utilizando a fórmula de Bhaskara encontramos duas raízes para a função, raízes úteis para a elaboração do gráfico dessa mesma função.
